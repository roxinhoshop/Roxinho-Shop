# 🛒 Roxinho Shop

E-commerce moderno de eletrônicos com design Fluent e tema escuro.

## 📁 Estrutura do Projeto

```
Roxinho-Shop/
├── src/
│   ├── recursos/          # Imagens, fontes e ícones
│   ├── estilos/          # CSS organizados por categoria
│   │   ├── base/
│   │   ├── componentes/
│   │   ├── layouts/
│   │   └── temas/
│   ├── scripts/          # JavaScript modular
│   │   ├── componentes/
│   │   ├── servicos/
│   │   ├── utilitarios/
│   │   └── configuracao/
│   └── paginas/          # Páginas HTML
├── publico/              # Arquivos públicos
├── documentacao/         # Documentação do projeto
└── configuracao/         # Arquivos de configuração
```

## 🚀 Tecnologias

- HTML5
- CSS3 (Fluent Design System)
- JavaScript (ES6+)
- Font Awesome
- Google Fonts

## 📦 Funcionalidades

- ✅ Sistema de autenticação
- ✅ Catálogo de produtos
- ✅ Filtros avançados
- ✅ Histórico de visualizações
- ✅ Tema escuro/claro
- ✅ Design responsivo
- ✅ Painel administrativo

## 🎨 Design

- Fluent Design System (Windows 11)
